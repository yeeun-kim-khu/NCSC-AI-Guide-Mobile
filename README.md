# 🐣 국립어린이과학관 AI 가이드

Streamlit Cloud 배포용 버전입니다.

## 🚀 Features

- 🤖 AI 챗봇 가이드 (GPT-4 기반)
- 🎤 음성 입출력 지원
- 🌏 다국어 지원 (한국어, English, 日本語, 中文)
- 📚 사후 학습 시스템 (퀴즈, 과학동화)
- 🔍 RAG 기반 전시물 검색

## 📦 Requirements

- Python 3.11+
- OpenAI API Key

## 🔧 Setup

Streamlit Cloud Secrets에 다음을 추가하세요:

```toml
OPENAI_API_KEY = "your-api-key-here"
```

`app_with_voice.py`
